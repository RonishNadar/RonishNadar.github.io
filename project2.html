Project

Dual Track Ackermann Drive Simulation

A physics-based simulation of a four-wheel, double-track Ackermann vehicle that integrates rigid-body dynamics, nonlinear control, and curvature-aware path planning. The system models realistic vehicle behavior and evaluates robust trajectory tracking under dynamic obstacles while maintaining real-time performance.

Vehicle Dynamics Ackermann (Double-Track) Sliding Mode Control Hybrid A* B-Splines Real-Time (100 Hz)

Simulation Demo

Highlights

  • Vehicle Dynamics: Implemented a four-wheel, double-track Ackermann model using Newton–Euler rigid-body equations with linear tire cornering stiffness for a 12 kg platform.
  • Sliding Mode Control: Designed a robust SMC for trajectory tracking with a smoothed tanh-based reaching law to reduce chattering.
  • Stability Analysis: Verified closed-loop stability using Lyapunov candidate functions, ensuring convergence of tracking error dynamics.
  • Planning & Smoothing: Generated curvature-feasible paths using Hybrid A* and B-spline interpolation for smooth steering and reduced control effort.
  • Real-Time Performance: Optimized the control loop to run at 100 Hz while enforcing steering and actuator constraints.

System Overview

The simulator follows a full autonomy pipeline: a planner produces curvature-aware paths, spline interpolation smooths geometric discontinuities, and a nonlinear adaptive controller tracks the reference trajectory under realistic vehicle dynamics and actuator limits.

Technical Breakdown

Vehicle Dynamics Modeling

Unlike simplified bicycle models, the simulator uses a double-track formulation that models left and right wheel pairs explicitly. Planar motion is derived from Newton–Euler equations, capturing longitudinal, lateral, and yaw dynamics. Tire forces are generated using linear cornering stiffness, enabling realistic slip-angle behavior while preserving numerical stability.

Sliding Mode Control Design

A sliding mode controller was developed to handle model uncertainty and disturbances. To mitigate high-frequency chattering, a smooth hyperbolic tangent reaching law replaces the discontinuous sign function. Closed-loop stability is demonstrated via Lyapunov analysis, guaranteeing convergence of tracking errors.

Hybrid A* Planning & Spline Smoothing

Hybrid A* generates dynamically feasible paths that respect nonholonomic Ackermann constraints and minimum turning radius. B-spline interpolation smooths curvature transitions, improving tracking performance and reducing steering oscillations under high-speed operation.

Key Takeaways

  • Demonstrates rigorous vehicle dynamics beyond kinematic bicycle models
  • Applies nonlinear control theory with formal stability guarantees
  • Integrates planning, smoothing, and control in a real-time loop
  • Bridges theory-heavy control design with practical autonomous driving constraints
