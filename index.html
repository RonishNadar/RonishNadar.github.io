Ronish Nadar

Robotics & Embedded Systems Engineer pursuing an M.S. in Mechatronics & Robotics at NYU. I build autonomous systems from the ground up writing embedded firmware to interface sensors/actuators and communications, then integrating perception, planning, and control for reliable real-world behavior. I'm especially interested in roles that sit at the intersection of robotics, embedded software, and system integration.

Key skills

  • Embedded & Firmware: C, C++; STM32, ESP32, Raspberry Pi, Jetson; UART, I2C, SPI, CAN, RS485, RS232, MODBUS; Wi-Fi, BLE, LoRa, LTE; Factory Test Automation; PCB Design (Altium, KiCad, Eagle)
  • Robotics & Controls: Path Planning (A*, D*, RRT/RRT*, Hybrid A*); SLAM; Navigation Stacks; PID, LQR, MPC; Pure Pursuit; System Identification
  • Perception, Estimation & AI: OpenCV, YOLO, ArUco/AprilTag Tracking; Kalman Filters; Sensor Fusion (IMU, LiDAR, Cameras, Encoders, GPS); VLM, LLM Tooling (Moondream, Gemini, Llama), RAG
  • Tools: Python, ROS2, Linux (Ubuntu), Git, Docker, PyTorch, Simulink, MATLAB, NumPy

Autonomous Multi-Robot Control via LLM/VLM-Driven Embodied AI Frameworks

Built a “zero-code” embodied AI framework where natural language instructions are converted into multi-robot task schedules, using a global vision pipeline (stitching + calibration + ArUco tracking), dynamic obstacle monitoring, and an RRT* + Pure Pursuit/PID control stack bridged through a low-latency RPC server.

Dual-Track Ackermann Drive Simulation

Built a physics-based 4-wheel Ackermann vehicle simulator with Newton-Euler rigid-body dynamics, a sliding mode controller for robust trajectory tracking, and Hybrid A* + spline-based planning for curvature-aware obstacle avoidance at 100 Hz.

TATA Power: 110 kV Insulator Cleaning Robot

Developed a high-voltage-safe robotic system featuring a 4-DoF SCARA manipulator, modular embedded hardware, and vision-based inspection pipelines, reducing routine inspection time by 3× and improving field efficiency by 40%.

Punch ‘n’ Pop: Rehabilitative Motion-Controlled Game

Built an accessibility-first rehab game that fuses OpenCV color tracking with MPU6050 IMU signals to detect multi-quadrant gestures and strike intensity, streaming sensor data over ESP-NOW for low-latency scoring, visualization, and session data sync.
